# line

a status line

currently shows:
* [i3](https://github.com/i3/i3) workspaces.
* alsa volume

## build

```shell
make
```

## usage

```shell
./line
```

![Screenshot](screenshot.png)

## Dependencies

* [i3ipc-glib](https://github.com/acrisci/i3ipc-glib)
* alsa
* xcb and xcb-randr
